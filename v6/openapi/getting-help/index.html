Getting help

  • consult the OpenAPI specs for a complete reference of the presented objects
  • see more examples of OpenAPI object representations in zircote/swagger-php's GitHub repository
  • consult zircote/swagger-php's online documentation or run the following command to see their help page:
./vendor/bin/openapi --help
